# Student-Resource-Locator
This project is being designed to be a platform where students and even faculty members can upload and download resources such as lecture notes, handouts, assignments and even past questions.
Here, downloading materials needed for each course across various levels across the department, faculty, or school will be made easily accessible anyday-anytime.
The platfom is user dependent, and all materials uploaded will first be examined by an admin to ensure quality control and prevent malicious content uploaded by malicious individuals.
